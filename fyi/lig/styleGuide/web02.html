  1. Document Type

    - Document Type 별로 css의 기능이 틀립니다.

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">개발후에도 항상 상위에 위치시킵니다.

  2. HTML head

    - Common.css 공통css

    - Print.css 프린트시 화면을 컨트롤하는 css입니다.

    - Common.js 공통 자바스크립트 함수를 호출합니다.

    - FlashX.js 플래시제어 스크립트를 호출합니다.(플래시 박스제거 및 외부링크 호출)

    <head>
    <title>:: LIG ::</title>
    <link rel="stylesheet" href="CSS/Common.css" type="text/css" media="all">
    <link rel="stylesheet" href="CSS/Print.css" type="text/css" media="print">
    <script type="text/javascript" src="JS/Common.js"></script>
    <script type="text/javascript" src="JS/flashX.js"></script>
    </head>

  4. CSS Structure

    - import CSS를 이용하여 가독성을 높이고 불필요한 css를 걸러냅니다.

    - Develop.css를 이용하여 개발부분에서 차이가 생긴부분만 개발자가 기재합니다.

    - @charset "utf-8"; 인코딩환경에 맞추어 작업

    @charset "utf-8"; == 개발부분에서 한글 부분을 %-escape 하지않아도 되기때문에 utf-8 사용

    @import "Public.css"; == import를 사용
    @import "Private.css";
    @import "Table.css";
    @import "Font.css";
